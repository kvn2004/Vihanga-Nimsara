# Vihanga Nimsara - Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS. Features a dark-mode-first design with glassmorphism effects, smooth animations, and interactive components.

##  Features

- **Modern Tech Stack**: React 19, TypeScript, Tailwind CSS, Framer Motion
- **Dark Mode First**: Professional dark theme with optional light mode support
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop
- **Interactive UI**:
  - Typing animation effect in hero section
  - Smooth scroll navigation
  - Project filtering by category
  - Hover animations and micro-interactions
- **Performance Optimized**: Code splitting and lazy loading
- **SEO Friendly**: Proper meta tags and semantic HTML
- **Accessible**: ARIA roles and keyboard navigation support
